Le projet entre dans le cadre de développement d’une application permettant l’exploitation d’information sur le salaire et la disparité salariale selon le genre.


L’application affiche un formulaire pour compléter les paramètres suivants :
  • Type d’emploi;
  • Classification nationale des professions (CNP4);
  • Sexe;
  • Région;
  • Groupe d’âge.
